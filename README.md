# war-diaries


## Run apps

- open a httpServer (e.g., python -m http.server) at the root (in the 'war-diaries' folder)


## File Structure

- root
  - README.md
  - dev
    - data
      - _Data files, organized by category then by date_
    - module
      - _Application modules_
    - lib
      - _Some useful libraries for offline development_
